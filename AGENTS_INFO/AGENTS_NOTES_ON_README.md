# AGENTS_NOTES_ON_README.md

## Purpose

This files describes how `README.md` should be structured and maintained in this repository.

The README is an operator guide, not a full internal design document.
It should staty concise, practical, and aligned with the current supported workflows.

## Audience

The primary audience is an operator or maintainer who needs to:

- understand what this repository is for
- prepare their environment correctly
- choose the right operator-facing utility
- run that utility
- understand the expected output
- maintain this repo
